# Steam Workshop Analyzer
It lets you know how much disk space each workshop mod you have on Steam takes

I made this project because I'm currently learning Rust, it works well on Windows and Linux, but you must have workshop mods installed.
It also lets you provide a Steam app id and it will show you the app name (e.g. 108600 -> Project Zomboid).

Any questions or ideas are welcome.

To run it just do ```cargo run``` or ```cargo build --release``` (if you want the exe file) in the project directory, you must have cargo installed: https://doc.rust-lang.org/cargo/getting-started/installation.html
